## Como Usar Esta Documentação

Esta documentação foi projetada para ser consultada de acordo com suas necessidades específicas:

- **Desenvolvedores Frontend**: Comece com as APIs e Endpoints para entender como integrar o frontend com o backend.
- **Desenvolvedores Backend**: Comece com os Modelos e Entidades e os Fluxos de Negócio para entender a arquitetura do sistema.
- **Testadores**: Consulte a seção de Testes para entender como testar o sistema.
- **DevOps**: Consulte o Guia de Configuração do Ambiente para configurar o ambiente de produção.
